# Module 2: Issue Structuring & Framing

Module 1 outputs issues, which this module uses as input.
Each issue's submissions are grouped based on what they are about.
This structured issue view is used by module 3.

## Inputs

**Issue**  
(from module 1)

## Outputs

**Structured Issue View**  
This is used as input for module 3.

## Functionality

### Submissions Within Issue are Clustered

Submissions are grouped together based on how similar they are to each other.

### Submission Clusters are Labeled with a Theme

A label is generated for each cluster based on the cluster's shared subject matter.
